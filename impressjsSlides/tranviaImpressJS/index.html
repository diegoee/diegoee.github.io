Introdución

Aplicación Android de fiscalización de tarjetas Bono Tranvía en un dispositivo de carácter especificado, Famoco fx100

Idea: Mejorar la rapidez de la lectura de tarjetas y su comodidad con un nuevo dispositivo más ligero y moderno

Requisitos y Objetivos

Dispositivo Android: Famoco fx100,7.

Leer tarjeta Bono Tranvía. Decodificar el mapa de bit disponible. Mifare Classic 1K.

Algunas tarjetas son decodificadas con un clave fija estática (Bono Tranvía Monocolor)

Otras tarjetas requieren una autenticación con el modulo SAM (Secure Access Module) para obtener la clave para la lectura de la tarjeta. (Bono Tranvía Bicolor o Tricolor)

Tecnologías

Harware:

NFC. Near Field Communication.

SAM. Secure Access Module

Tarjetas Mifare 1k Classic. 16 sectores de 4 bloques cada uno y cada bloque pose 16 bytes de información. (16 Bloques x 4 Sectores x 16 bytes=1024 bytes = 1Kb). tecnología propiedad de NXP Semiconductors.

Software:

Android Studio.
